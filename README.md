# SailZ

**שלבים**
מציאת קורדינטות של מוצא ויעד
מציאת מסלול ימי הכי קצר
אינפורמציה של מזג האוויר על נקודות בדרך
עיבוד המידע לכדי המלצות
שיקוף ויזואלי של המסלול והמלצות
----------------------------

**טכני**
API למציאת קורדינטות לעיר
פונקציה ליצירת רשת ומציאת מסלול
API למידע רלווונטי על הנקודות
פונקציה לעיבוד המידע להמלצות
קליטת המידע בdf
שמירה בcsv
יצירת דף HTML & Streamlit
----------------------------

**אתגרים**
מיפוי ימי- 
שימוש ברשת מחוברת בנקודות 
כיול אופטימאלי למפה
----------------------------

**תקלות שטופלו**
מיקום שגוי - 
שורש הבעיה החלפה של קו אורך רוחב במעבר בין פונקציות

חצית יבשות - 
נפתר על ידי דיוק הרשת

חסימה API עקב ריבוי בקשות למסלולים רחוקים- 
החזרה של מספר נקודות מוגבל מתוך המסלול בהתפלגות שווה.
